# Steps For Run
**Create .env file and paste given env**


**For run node js server**


``` cd server ```
``` npm i ```
``` nodemon server.js ```


**For run react server**


``` cd client ```
``` npm i ```
``` npm run dev ```

2. use cd command to change directory !
